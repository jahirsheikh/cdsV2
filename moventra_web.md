# CDS Website Requirements Specification (Static Website)

## Project Name

**CDS**

---

# Project Overview

Develop a modern, professional, fully responsive static corporate website for **CDS** inspired by the overall structure, user experience, and functionality of **https://itracker.com.bd**.

The website should not use any backend, database, CMS, or API. All contents will be static and stored locally.

The design should reflect a modern software company specializing in GPS Tracking, Fleet Management, IoT Solutions, Vehicle Security, Asset Tracking, and Enterprise Digital Solutions.

# Notes

* The website should take **https://itracker.com.bd** as inspiration for information architecture and user experience, while featuring a completely original visual identity and codebase.
* All content, images, icons, and graphics should be royalty-free or represented with placeholders that can be replaced later.
* No backend, database, authentication, payment gateway, or API integration is required.
* Every page and component should be fully responsive and optimized for desktops, tablets, and mobile devices.
* The final design should establish **CDS** as a premium technology brand with a strong focus on GPS tracking, fleet management, IoT solutions, and enterprise digital services.
