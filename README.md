# CDS Static Website

This project is a modern, responsive static corporate website for CDS, a company specializing in GPS Tracking, Fleet Management, and IoT Solutions.

## Features
- Fully responsive design
- Modern UI/UX with smooth animations
- Built with HTML5, CSS3, JavaScript
- Uses Bootstrap 5.x for layout and utility classes
- Icons from Font Awesome
- Scroll animations using AOS
- No backend required

## Setup & Running
Since this is a static website, you can run it simply by opening `index.html` in any modern web browser.

No build tools or servers are required.

## Folder Structure
- `assets/css/` - Contains all stylesheets including variables and responsive rules.
- `assets/js/` - Contains main JavaScript logic.
- `index.html` - Home Page
- `about.html` - About Us Page
- `contact.html` - Contact Page
